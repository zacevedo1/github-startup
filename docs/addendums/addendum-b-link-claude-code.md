# Addendum B: Open This Folder in Claude Code

Use this if Claude Code is the AI coding tool you chose for the tutorial.

## Goal

Open the same local project folder in Claude Code.

## Instructions

1. Open Claude Code or its overview page: <https://code.claude.com/docs/en/overview>
2. Open or select this folder:

   `Documents/GitHub/github-startup`

3. Ask Claude Code to list the files it can see.
4. Confirm it can see `README.md`, `index.html`, and `style.css`.

If Claude Code on your computer requires Terminal and you are not comfortable with Terminal, see `addendum-e-opencode-and-deepseek.md` for a free path that does not require Terminal.
