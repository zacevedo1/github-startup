# Addendum A: Open This Folder in Codex

Use this if Codex is the AI coding tool you chose for the tutorial.

## Goal

Open the same local project folder in Codex.

## Instructions

1. Open Codex: <https://chatgpt.com/codex/>
2. Open or select this folder:

   `Documents/GitHub/github-startup`

3. Ask Codex to list the files it can see.
4. Confirm it can see `README.md`, `index.html`, and `style.css`.

Keep using GitHub Desktop for commit and push.
