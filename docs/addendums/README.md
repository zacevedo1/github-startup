# Tool Notes and Optional Addendums

The required tutorial path uses GitHub Desktop and one AI coding tool. The tier model is:

- **Free:** opencode (with a free vision-capable model in its picker). See addendum E.
- **Paid:** Codex (requires ChatGPT Pro) or Claude Code. See addendums A and B.

You only need one AI coding tool to finish the tutorial. The addendums below are optional notes for each tool plus a couple of other side-paths.

A. [Open This Folder in Codex](addendum-a-link-codex.md) - <https://chatgpt.com/codex/>
B. [Open This Folder in Claude Code](addendum-b-link-claude-code.md) - <https://code.claude.com/docs/en/overview>
C. [GitHub CLI / Terminal Authentication](addendum-c-github-cli-login.md) - advanced only
D. [Open This Folder in VS Code](addendum-d-open-in-vscode.md) - optional editor only
E. [Use opencode and the Free Vision Model](addendum-e-opencode-and-deepseek.md) - <https://opencode.ai/> - free path with vision
F. [Edit Files in the Browser with github.dev](addendum-f-github-dev-browser-edit.md) - optional, for quick browser-only edits
