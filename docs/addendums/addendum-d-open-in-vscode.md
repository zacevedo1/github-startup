# Addendum D: Open This Folder in VS Code

VS Code is optional for this tutorial. The required path uses opencode, Codex, or Claude Code for AI help and GitHub Desktop for commit/push.

## Goal

Open the same local folder in VS Code if you want another code editor.

## Instructions

1. Install VS Code: <https://code.visualstudio.com/>
2. Open VS Code.
3. Choose **File > Open Folder**.
4. Select:

   `Documents/GitHub/github-startup`

5. Confirm you can see `README.md`, `index.html`, and `style.css`.

Keep using GitHub Desktop for commit and push.
