# Publish Your Dashboard with GitHub Pages

## Goal

Turn your repository into a live website so your dashboard has a public URL.

## Instructions

1. On GitHub in your browser, open your `github-startup` repository.
2. Click the **Settings** tab.
3. In the left sidebar, click **Pages**.
4. Under **Branch**, select `main`.
5. Keep the folder as `/ (root)`.
6. Click **Save**.
7. Wait 1-2 minutes.
8. Refresh the page -- a blue banner will appear with your live URL.

Your link will look like:

`https://your-username.github.io/github-startup/`

Open that URL in a new tab. You should see your dashboard live on the web.

## Note

You came into this step with a dashboard already built by your AI tool. The first thing visitors see at your live URL is your real work, not a blank starter template.

From now on, every time you commit and push a change, the live page will update automatically within a few minutes. No extra steps needed.

## Checklist

- [ ] GitHub Pages is enabled (source = `main`, folder = `/ (root)`).
- [ ] I waited 1-2 minutes after enabling Pages.
- [ ] I opened my live GitHub Pages link in a browser.
- [ ] My dashboard renders correctly at the live URL.
- [ ] I have the live URL ready to submit on Canvas.
