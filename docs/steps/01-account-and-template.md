# Start Here: Create Your GitHub Account, Tutorial Copy, and Step Issues

## Goal

Create your own online copy of this starter project and generate step-by-step issues.

The README shows this same start sequence. This file is kept as a reference, but the Action does not create an issue from it.

## Instructions

1. Create a GitHub account if you do not already have one.
2. Open the original `github-starter-tutorial` repository.
3. Click **Use this template**.
4. Choose **Create a new repository**.
5. Name your new repository `github-startup`.
6. Create it under your own GitHub account.
7. Open your new `github-startup` repository.
8. Click the **Actions** tab.
9. If GitHub asks you to enable workflows, enable them.
10. Click **Initialize Tutorial Issues**.
11. Click **Run workflow**.
12. Wait for the workflow to finish.
13. Open the **Issues** tab.
14. Use the generated issues as your step-by-step checklist.

Do not click the green **Code** button yet. You will use that later with GitHub Desktop.

## Checklist

- [ ] I have a GitHub account.
- [ ] I created my own `github-startup` repository from the template.
- [ ] I can see `github-startup` under my own GitHub account.
- [ ] I ran **Initialize Tutorial Issues** from the Actions tab.
- [ ] I can see the generated step issues in the Issues tab.
