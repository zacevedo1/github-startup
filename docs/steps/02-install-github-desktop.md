# Install GitHub Desktop and Clone Your Repo

## Goal

Copy your online repo to your computer so you can build your dashboard with an AI coding tool.

## Instructions

### Part A: Install GitHub Desktop

1. Go to https://desktop.github.com and download GitHub Desktop.
2. Open the downloaded file and install it (drag to Applications folder on Mac, or run the installer on Windows).
3. Open GitHub Desktop.

### Part B: Sign in

1. In GitHub Desktop, click **Sign in to GitHub.com**.
2. Your browser will open. Sign in to your GitHub account and authorize the connection.
3. Go back to GitHub Desktop. You should see your account name.

### Part C: Clone your repo

1. On GitHub in your browser, open your `github-startup` repository.
2. Click the green **Code** button.
3. Choose **Open with GitHub Desktop**.
4. GitHub Desktop will open and ask where to save the folder.
5. Set the location to `Documents/GitHub/github-startup`.
6. Click **Clone**.

Your local folder is now at:

`Documents/GitHub/github-startup`

## Checklist

- [ ] GitHub Desktop is installed.
- [ ] GitHub Desktop is signed in to my GitHub account.
- [ ] I cloned `github-startup` into `Documents/GitHub`.
- [ ] GitHub Desktop shows `github-startup` as the current repository.
