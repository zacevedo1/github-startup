# Commit and Push from GitHub Desktop

## Goal

Save your AI-built changes and send them up to GitHub so they can be published.

## Instructions

1. Open GitHub Desktop.
2. Make sure the current repository is `github-startup`.
3. Look at the changed files list on the left. Click a file to see what changed.
4. In the **Summary** box at the bottom-left, write a short message like `Build research dashboard`.
5. Click **Commit to main**.
6. Click **Push origin** (appears after you commit).

Your changes are now on GitHub. The next step is to publish them as a live website.

## Your workflow from now on

Once you have done this once, the pattern repeats every time you make a change:

1. Make changes in your AI coding tool (or directly in the files).
2. Open GitHub Desktop.
3. Write a summary and click **Commit to main**.
4. Click **Push origin**.
5. Once Pages is published, your live page updates automatically within a few minutes.

## Checklist

- [ ] I committed my changes in GitHub Desktop.
- [ ] I pushed to GitHub.
- [ ] I can see my changes on the GitHub repo page in the browser.
